元素类别及标签嵌套规则

元素类别

block 块级元素

功能 ：用来搭建结构，承载内容，布局
eg:div,ul,ol,dl,li,h1~h6,p

特点：

  1. 独占一行
  2. 默认宽度为父级的100%
  3. 可以直接设置宽高
  4. 对margin,padding有效

inline 行内元素

功能：用来区分，强调，细节化
eg:span,strong,em…,a

特点：

  1. 横排排列，直到装不下才会换行
  2. 不能直接设置宽高(内容撑开width height)
  3. 对垂直方向上的margin,padding无效

inline-block 行内块级元素

特点：

  1. 既能设置宽高，又能在同一行中排列

行内替换元素：img input

标签嵌套规则

  1. 块级元素可以嵌套大部分块级和行内元素
  2. 行内元素不能套块级，可以套行内
  3. 特殊的h1~h6,p,dt,不能套块级，只能套行内
  4. 同级的元素类别要相同
